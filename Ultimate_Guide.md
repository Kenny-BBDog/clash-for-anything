# 🚀 Clash For Anything: 终极使用手册

这套系统设计的初衷是：**隐私归于本地，性能归于 VPS，操作归于傻瓜。**

---

## 阶段一：一劳永逸的初始化 (仅需做一次)

### 1. 部署你的“性能中转站” (VPS)
登录你的 VPS（如 DMIT 等 1Gbps 以上的服务器），运行以下命令：
```bash
bash <(curl -Ls https://raw.githubusercontent.com/Kenny-BBDog/clash-for-anything/main/vps-setup/setup.sh)
```
- **它会帮你做什么？** 开启 BBR 提速、优化 TCP 缓冲区、安装 3x-ui 面板，以及最关键的——**安装私有转换引擎 (Subconverter)**。
- **注意**：脚本运行完后，请记下你的 **VPS 公网 IP**。

### 2. 搭建你的“私人控制面板” (GitHub Pages)
1. Fork 本仓库到你自己的 GitHub 账号下。
2. 进入仓库的 `Settings` -> `Pages`。
3. `Source` 选 `Deploy from a branch`，`Branch` 选 `main`，路径 `/ (root)`，点击 `Save`。
4. 几分钟后，你会得到一个专属网址：`https://你的用户名.github.io/clash-for-anything/`。

---

## 阶段二：日常魔法操作 (节点更新/首次导入)

当你买了新节点或想更新配置时：

1. **打开你的私人网页**：访问上面生成的网址。
2. **填入节点**：把你的 VLESS/VMess 链接全部粘贴进左侧大框。
3. **(可选) 启用住宅 IP**：
   - 点击右侧“启用本地住宅代理”。
   - 填入你的 IPRoyal 账号密码。
   - **落地转发**：在下拉框选一个你刚才贴进去的 VPS 节点名（让住宅 IP 流量通过你的 VPS 转发，速度更稳）。
4. **生成订阅**：
   - 在“私有订阅链接”框里填入你的 **VPS IP**。
   - 点击 **“📋 复制在线订阅链接”**。
5. **导入 Clash**：
   - 打开 Clash Verge (Windows) 或 Clash Meta (Android)。
   - 新建订阅 -> 粘贴刚才复制的 URL。
   - **完成！** 现在你软件里的规则（微信直连、AI 走住宅 IP）都是最新的。

---

## 阶段三：更换住宅 IP (热插拔)

**这是系统的核心优势：下个月换 IP 账号时，不需要再去管代码。**

1. 直接打开网页。
2. 在右侧“住宅 IP 模块”里直接把旧的账号密码删掉，填入新的。
3. 点击 **“📋 复制在线订阅链接”**（确保参数已更新）。
4. 在 Clash 软件里右键你的订阅 -> 点击 **“更新 (Update)”**。
5. **瞬间生效**。AI 流量会立刻无缝切换到新的住宅 IP 上，不需要重启，不需要改配置文件。

## 💡 常见问题 Q&A

- **Q: 为什么生成的链接地址是 http://我的IP:25500/...？**
  - A: 因为这是直接请求你 VPS 上的转换引擎，不经过任何第三方，绝对保护你的节点隐私。
- **Q: 网页里的信息刷新会丢吗？**
  - A: 不会。信息保存在你浏览器的 `localStorage` 里，除非你手动清除浏览器缓存。即便你把 GitHub 仓库开源了，别人打开网页也看不到你的这些本地数据。
- **Q: AI 真的会自动走住宅 IP 吗？**
  - A: 是的。只要你开启了住宅 IP 模块并选择了落地节点，系统生成的规则会自动拦截 ChatGPT/Claude 等域名并导向该通道。
